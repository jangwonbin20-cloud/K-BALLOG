# K-BALLOG 프로젝트 청사진

## 1. 프로젝트 개요

K-BALLOG는 대한민국 아마추어 축구 리그의 소식을 통합 제공하는 웹 플랫폼입니다. 사용자는 지역별(서울, 수원, 인천, 용인 등) 리그의 경기 일정, 결과, 순위, 관련 뉴스 등 다양한 정보를 손쉽게 확인하고, 자신이 응원하는 팀과 선수들의 소식을 접할 수 있습니다.

## 2. 디자인 시스템 및 레이아웃

### 2.1. 시각적 정체성

- **테마**: 전체적으로 어두운 테마(Dark Mode)를 적용하여 모던하고 집중도 높은 사용자 경험을 제공합니다.
- **컬러 팔레트**: 주조색으로 `--dark-bg` (`#1a1c23`), 카드 배경색으로 `--card-bg` (`#24262e`), 그리고 강조색으로 강렬한 빨간색 계열의 `--primary-accent` (`#e4002b`)를 사용합니다.
- **아이콘**: Font Awesome 아이콘을 활용하여 메뉴와 버튼의 시각적 이해를 돕습니다.
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에 대응하는 반응형 레이아웃을 갖추고 있습니다.

### 2.2. 핵심 컴포넌트: 통합 카드 (`.unified-card`)

뉴스, 리그 바로가기, 팀 목록 등 다양한 콘텐츠를 일관된 형태로 표시하기 위한 재사용 가능한 카드 컴포넌트입니다. 이미지, 제목, 설명 텍스트로 구성되며, 마우스를 올리면 부드러운 애니메이션 효과가 적용되어 사용자 인터랙션을 유도합니다.

### 2.3. 페이지 레이아웃

- **메인 콘텐츠 + 사이드바 레이아웃**: 주요 정보를 담는 넓은 메인 콘텐츠 영역과 추가 정보(바로가기 링크, 매치 센터 등)를 제공하는 좁은 사이드바 영역으로 구성된 2단 레이아웃을 사용합니다. (`seoul-k7.html`, `seoul-k7-seongbuk.html`)
- **그리드 레이아웃**: 여러 개의 통합 카드를 격자 형태로 배열하여 보여주는 데 사용됩니다. 화면 크기에 따라 열의 개수가 자동으로 조절됩니다. (`index.html`, `seoul.html` 등)

## 3. 개발 이력 및 변경 사항

### 3.1. 초기 디자인 개편 (1차)

- **목표**: 분산되어 있던 콘텐츠 표시 방식을 통일하여 사용자 경험의 일관성을 확보합니다.
- **변경 사항**:
    1.  **통합 카드 도입**: `.unified-card` 스타일을 `style.css`에 정의하여 사이트 전반의 카드 디자인을 통일했습니다.
    2.  **홈페이지(`index.html`) 구조 변경**: 여러 섹션을 '지역별 리그 바로가기'로 통합하고, 통합 카드를 사용한 그리드 레이아웃으로 개편했습니다.
    3.  **지역 페이지(`seoul.html`) 리팩토링**: 하위 리그 링크를 통합 카드 디자인으로 교체하여 통일성을 강화했습니다.

### 3.2. K7 리그 페이지 디자인 고도화 (2차)

- **목표**: K7 리그 관련 페이지의 정보 전달력을 높이고, 시각적으로 더욱 매력적인 페이지를 구성합니다.
- **변경 사항**:
    1.  **`seoul-k7.html` 디자인 적용**: 
        - 페이지 상단에 대표 이미지와 리그 소개를 담은 `.featured-article` 컴포넌트를 추가했습니다.
        - 메인 콘텐츠와 사이드바(바로가기 링크)로 구성된 2단 레이아웃을 적용했습니다.
        - 각 권역별 리그(강북, 성북 등) 목록을 이미지와 설명이 포함된 통합 카드로 표시하여 시각적 완성도를 높였습니다.
    2.  **`seoul-k7-seongbuk.html` 디자인 통일**:
        - `seoul-k7.html`과 동일한 레이아웃을 적용하여 일관성을 유지했습니다.
        - '참가팀'과 '주요 뉴스' 섹션을 통합 카드를 사용한 그리드 형태로 재구성하여 가독성과 시각적 매력을 향상시켰습니다.
        - 사이드바에 '순위 보기', '경기 일정 보기' 등의 바로가기 링크를 추가하여 사용자 편의성을 증대시켰습니다.

### 3.3. 내비게이션 확장 및 신규 페이지 생성 (현재)

- **목표**: 뉴스 및 커뮤니티 기능을 위한 기반을 마련하고, 모든 페이지에서 일관된 내비게이션을 제공합니다.
- **변경 사항**:
    1. **내비게이션 메뉴 추가**: `index.html`을 포함한 모든 주요 페이지의 헤더에 '뉴스'와 '커뮤니티' 메뉴 링크를 추가합니다.
    2. **`news.html` 페이지 생성**: 뉴스 기사 목록을 보여주는 페이지를 생성합니다.
    3. **`community.html` 페이지 생성**: 사용자 커뮤니티 게시판 페이지를 생성합니다.
